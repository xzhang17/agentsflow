# Legacy Agents Flow v2 Profile Library

This path is retained only so versioned v2 workflow snapshots fail clearly instead of resolving to unrelated current content.

Agents Flow 3.0 separated the former mixed library into:

- `references/profiles.md` — task obligations and validation profiles;
- `references/execution-modes.md` — checklist implementation modes and review routing.

Do not use this file for a new run. A frozen workflow that requires the v2 `modes.md` contract must be regenerated under Agents Flow 3.0. The old snapshot itself remains immutable.
