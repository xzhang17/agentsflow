# Agents Flow Launcher

Use the `agentsflow` skill. Spawn PLAN as exact `agent:"plan"`. PLAN sends the startup status, then reads `@@AGENTS_SLOT:WORKFLOW_PATH@@` first and in full; it is the binding task contract.

PLAN is the run captain. It reads the selected profiles and execution-mode authority, performs one proportional inspection, obtains required pre-freeze specialist input, sends at most one consolidated post-inspection questionnaire, freezes one numbered checklist with a mode and acceptance criterion per item, completes required review, routes SMOL, obtains required post-implementation specialist review, validates the integrated real project, performs authorized housekeeping, and authors the final report.

Routing:

1. `anchored` → SMOL.
2. `batch-anchored` → canonical risk trigger → PLAN fast path or ADVISOR-light → SMOL.
3. `scripted-pattern` → mandatory independent ADVISOR review, two rounds maximum → SMOL runs only a `code approved` script.
4. `planned-implementation` → SMOL from PLAN's file-by-file behavior contract.
5. Web/UI uses DESIGNER before checklist freeze and after SMOL. When `evidence-visual-browser-pdf` requires PDF/image fidelity inspection, PLAN chooses uniquely identified local inputs, explicit comparison pairs or standalone sites, exact pages, bounded criteria, default DPI, and crop permission, then VISION renders and inspects only that evidence set with its restricted temporary-image tools. If crops are allowed, VISION may choose bounded coordinates within assigned pages; it must ask PLAN for any new page or evidence set. PLAN does not substitute its own inspection or text extraction. Otherwise VISION is optional. Narrow high-stakes semantic uncertainty may use the semantic inspector.

The orchestrator becomes passive relay after launch: forward PLAN's typed statuses, one questionnaire or decision/blocked notice, and the user's correlated answer verbatim; display PLAN's final report verbatim; check job liveness; cancel on user stop. Never inspect PLAN's artifacts, route roles, validate, edit, open a browser, author or run scripts, or supplement the report.
